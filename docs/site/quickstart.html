Quick start

This mirrors matplotlib's own quick start guide — if you know one, you know the other.

Get it

# In your CMakeLists.txt
include(FetchContent)
FetchContent_Declare(graphlib
    GIT_REPOSITORY https://github.com/Nosenzor/graphlib.git
    GIT_TAG v0.7.0)
FetchContent_MakeAvailable(graphlib)
target_link_libraries(your_app PRIVATE graphlib::graphlib)

A simple example

graphlib graphs your data on Figures, each of which contains one or more Axes. The simplest way to make a figure with axes is the pyplot facade:

#include <graphlib/pyplot.hpp>
namespace plt = graphlib::pyplot;

int main() {
    plt::plot(std::vector<double>{1, 2, 3, 4},
              std::vector<double>{1, 4, 2, 3});
    plt::savefig("first.svg");
}

Styling with format strings and kwargs

matplotlib's fmt shorthand works verbatim; keyword arguments become a designated-initializer struct with the same names and defaults:

plt::plot(x, y1, "r--o");                              // fmt shorthand
plt::plot(x, y2, {.linewidth = 2.0, .label = "phase"}); // kwargs style
plt::legend();

Subplots, images, fields

graphlib::Figure& fig = plt::figure({.figsize = {{8.0, 3.6}}});
auto axes = fig.subplots(1, 2);
axes[0][0]->plot(x, y);
axes[0][1]->imshow(data, rows, cols, {.cmap = "viridis"});
fig.tight_layout();

scatter, bar, hist, errorbar, pcolormesh, contourf, colorbar — all present with matplotlib semantics; see the gallery.

Math, dates, annotations

ax.set_title("$E = mc^2$");                 // mathtext in any label
ax.xaxis_date();                            // AutoDateLocator + ConciseDateFormatter
ax.annotate("peak", {x0, y0},
            {.xytext = {{x1, y1}},
             .arrowprops = graphlib::ArrowProps{.arrowstyle = "->"}});

Saving

plt::savefig("figure.svg");                 // vector, byte-deterministic
plt::savefig("figure.png", {.dpi = 300});   // AGG raster at any dpi
plt::savefig("figure.pdf");                 // vector + selectable text, LaTeX-ready

Interactive windows

Configure with -DGRAPHLIB_INTERACTIVE=ON (fetches GLFW), then plt::show(): drag to pan, scroll to zoom, h home, s save, q quit. FuncAnimation drives live plots.
